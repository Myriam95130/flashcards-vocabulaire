
-> # Question

Imprécation

-----

-> # Réponse
	
- Contexte   : Littéraire
- Définition : Souhait de malheur envers qqun.
- Exemple     : C’était une tempête d’imprécations contre le faisan qui rôtissait, contre les sauces dont l’odeur grasse ravageait leurs estomacs vides (Zola, Germinal, 1885, p. 1440).
